# Fresh-Groceries
You are a team of four AI agents working together to build a complete grocery selling application.
The goal is to create a functional app where users can browse grocery items, add them to a cart,
and place orders. The work should be divided clearly among the four agents, and each agent should
focus only on their assigned responsibilities while collaborating with the others. Agent 1 is
responsible for defining the product requirements and planning the system. This agent should
identify the core features of the grocery app, design the overall system structure, define the data
structure for products, carts, and orders, and outline the API structure and folder organization so
that the development process is clear and organized for the other agents. Agent 2 is responsible
for building the backend logic of the application. This agent should create all the necessary APIs for
managing products, cart operations, and order placement, implement the data models for
products, carts, and orders, and ensure that the server correctly handles requests and responses
while keeping the code modular and clean. Agent 3 is responsible for building the user interface of
the application. This agent should create the pages needed for the app such as product listing,
product cards, cart view, and checkout page, and implement the functionality that allows users to
browse grocery items, add items to the cart, view the cart contents, and place orders. Agent 4 is
responsible for integrating everything together and ensuring the application runs successfully. This
agent should connect the frontend with the backend, verify that all features work properly,
organize the final project structure, and ensure that the entire application can run locally. The final
output should be a working grocery application along with a localhost link where the app can be
accessed and tested..
